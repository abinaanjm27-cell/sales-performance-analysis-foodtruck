# sales-performance-analysis-foodtruck
Sales performance analysis of food truck transaction data — Google Sheets — September 2025
<br>
Food Truck Sales Analysis

Project Overview

Analyzed one month of food truck transaction data to identify high-performing menu items, low-performing products, customer preferences, and opportunities for menu optimization.

Dataset

- Period: September 2025
- Operating Days: 18
- Menu Items: 65
- Total Units Sold: 254
- Fields: Date, Item Name, Quantity

Objectives

- Identify best-selling menu items
- Detect underperforming products
- Analyze customer purchasing patterns
- Provide data-driven business recommendations

Tools Used

- Google Sheets
- SUMIF Functions
- Pivot Analysis
- Data Visualization

Methodology

1. Cleaned and structured raw transaction records.
2. Aggregated sales quantities by menu item.
3. Ranked products based on sales performance.
4. Visualized results using bar charts.
5. Applied Pareto (80/20) analysis to identify key revenue-driving products.
6. Developed recommendations based on observed sales patterns.

Key Insights

| Item | Units Sold |
|------|------------|
| Chicken Burger | 17 |
| Shawarma Roll | 13 |
| Mutton Galouti | 13 |
| French Fries | 12 |
| Chicken Wrap | 11 |
| Chicken Kebab Roll | 11 |

The top 10 menu items generated approximately 46.5% of total sales.

Menu Performance

- 32 out of 65 menu items sold only one unit during the analysis period.
- Nearly half of the menu contributed a relatively small portion of total sales.
- Several products showed consistently low customer demand.

Customer Preference

- Chicken-based products represented the strongest-performing category.
- Multiple chicken items ranked among the highest-selling products.

Recommendations

- Reduce menu complexity by reviewing low-demand products.
- Focus inventory planning around consistently high-selling items.
- Prioritize marketing efforts for proven best sellers.
- Test future menu additions within high-demand product categories.
